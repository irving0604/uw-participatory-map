# UW Coffee Shops Map

A participatory mapping project for UW students to share coffee shops near campus.
Users can click the map to add a coffee shop, recommend a favorite order, and leave a short note
about the vibe (study-friendly, outlets, noise level, etc.).

## Live Map
https://irving0604.github.io/uw-participatory-map/

## How it works
- Frontend: GitHub Pages (docs/index.html)
- Backend: Node.js API on Heroku
- Database: PostgreSQL (Heroku Postgres)
- Data collected: contributor name, coffee shop name, favorite order, message, and location
